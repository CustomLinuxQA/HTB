# Параллельный сетевой скан Docker API (2375)

## Что это такое (кратко)
Этот скан проверяет **все хосты в подсети /24** на наличие **открытого Docker Remote API (порт 2375)**  
без аутентификации.  
Открытый `2375` = **полный контроль над Docker-хостом** (часто root).

---

## Базовая команда

```bash
for i in $(seq 1 254); do
  (curl -s --connect-timeout 1 http://192.168.65.$i:2375/version 2>/dev/null \
    | grep -q "ApiVersion" && echo "192.168.65.$i:2375 OPEN") &
done
wait

Почему используется /version
Docker Remote API всегда отвечает на /version, например:
```bash
{
  "ApiVersion": "1.41",
  "Version": "20.10.12"
}

Наличие ApiVersion = это точно Docker.
Это значит:

- Docker API доступен по HTTP
- нет TLS
- нет аутентификации
- можно управлять Docker удалённо

##Почему это критично
- Открытый Docker API позволяет:
- запускать контейнеры
- монтировать / хоста внутрь контейнера
- получить root-доступ к системе
